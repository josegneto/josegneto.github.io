Baseei-me num projeto para realizar este guide: Teensy/Arduino – Memory Game.


1. Objetivo

Neste projeto utilizaremos uma matriz LED 8x8 e programamo-la de forma a acender os LEDs que escolhermos.
As possibilidades de trabalho que a matriz possibilita são inúmeras:

  • Enviar inputs para o Arduino através de um teclado e a matriz apresenta os caracteres recebidos;
  • Definir sequências que a matriz exibirá repetidamente;
